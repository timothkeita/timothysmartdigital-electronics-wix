# Project Title

**Timothy Smart Digital & Electronics – No-Code E-Commerce Website**

# Platform Used

This project was developed using **Wix**, a no-code website development platform. Wix was chosen because it provides powerful tools for creating professional e-commerce websites without writing code.

# Project Overview

Timothy Smart Digital & Electronics is an online electronics store that offers quality digital devices and accessories at affordable prices. The website provides customers with a simple and user-friendly shopping experience.

Products available on the website include:

* Mobile Devices
* Computer Accessories
* Digital Software

# Features Implemented

* Professional Homepage
* Online Product Catalog
* Product Categories
* Shopping Cart
* Product Images
* Product Prices
* About Us Page
* Contact Page
* Business Logo
* Responsive Navigation
* Customer Login
